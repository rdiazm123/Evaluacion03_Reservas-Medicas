Sobre la Aplicación

Sistema de Reservas Médicas desarrollado por Ruth & Romané

Angular 18

Framework frontend moderno basado en componentes standalone y rutas dinámicas.

Bootstrap 5

Librería de estilos para diseño responsivo, controles de formulario y componentes UI oscuros y llamativos.

LocalStorage

Almacenamiento persistente en el navegador web para guardar las reservas sin necesidad de una base de datos externa.

Objetivo del Proyecto

Esta aplicación demuestra el uso de rutas, formularios con validaciones avanzadas, almacenamiento local persistente y navegación en Angular moderno. Desarrollado de forma colaborativa para la Evaluación 3 del curso de Programación.
